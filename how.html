1
Скопируй конфиг

Ссылка скопируется автоматически — ничего больше делать не нужно.
2
Скачай и открой v2rayTun
Нажми кнопку Скачать v2rayTun выше.
Установи приложение как обычно и открой его.
3
Нажми + в правом верхнем углу
В приложении найди кнопку + — она находится в правом верхнем углу экрана.
Нажми на неё.
4
Выбери «Импорт из буфера обмена»
Появится меню. Выбери пункт Импорт из буфера обмена.

5
В списке появится новое соединение
Ты увидишь строчку с названием — это твоё личное подключение.
Нажми на неё, затем нажми большую кнопку Старт.
6
Разреши VPN подключение

Нажми Разрешить. Это обязательный шаг — без него не работает.

7
Готово — ты в эфире!
В строке статуса появится значок ключа 🔑 — это значит подключение активно.

Пошаговая инструкция
1
Скопируй конфиг
Нажми кнопку СКОПИРОВАТЬ КОНФИГ вверху этой страницы.
Ссылка скопируется автоматически.
2
Скачай и открой OneXray
Нажми кнопку Скачать OneXray выше.
Установи приложение и открой его.
Приложение на английском — это нормально, всё интуитивно понятно.
💡 Если App Store предложит выбрать регион — выбери любой, это не важно
3
При первом запуске — разреши всё
Приложение попросит несколько разрешений:
• Уведомления — нажми Allow (Разрешить)
• Доступ к сети — нажми Allow
Это стандартные запросы для таких приложений.
4
Нажми + в правом верхнем углу
Найди кнопку + в правом верхнем углу экрана и нажми её.
5
Выбери Import from clipboard
Появится меню. Выбери пункт Import from clipboard
(по-русски: Импорт из буфера обмена).
Приложение само возьмёт скопированную ссылку.
💡 Если пункт не появился — вернись на шаг 1 и снова скопируй конфиг
6
Включи переключатель рядом с соединением
В списке появится новая строчка — твоё подключение.
Нажми переключатель рядом с ней чтобы включить.
7
Разреши VPN подключение
iPhone покажет запрос — «OneXray Would Like to Add VPN Configurations»
Нажми Allow (Разрешить).
Телефон может попросить Face ID или пароль — это нормально.
🔒 Это стандартный запрос iOS. Без него VPN не работает ни в одном приложении.
8
Готово — ты в эфире!
В верхней строке телефона появится надпись VPN — подключение активно.
Теперь можно открывать любые сайты.
